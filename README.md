  # Контрольная работа по предмету "Разработка программных комплексов для исследований в химии и химической технологии"
  ## Вариант 3

  Инструкция запуска:

  1. Скачать и установить интерпретатор python.
     
          https://www.python.org/downloads/release/python-3108/
     
  2. Поставить необходимые библиотеки командой

         pip install -r requirements.txt

     
  3. Запуск

     
         python manage.py runserver
